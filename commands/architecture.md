Manage Architectural Specifications.

Delegate to the `axe-method:architecture-specifier` agent to explore and draft Architecture specifications for the target domain.

The agent covers system structure, bounded contexts, aggregates, context maps, boundaries, seams, layers, composition rules, state distribution, threading model, and dependency policy. It preloads the `axe-method:axe-method` skill and uses project-level persistent memory.
